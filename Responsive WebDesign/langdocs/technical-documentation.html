variables

Variables are the main part of coding. They are the elements that keep some important information in your code, such as the example below.

1: var x = 5

In the code above, the variable "x" stores the number 5, that can have many different uses on your file, such as math problems, info validation and more!
But you can have other kinds of variables:

  • Number
  • String
  • Boolean

if statement

If statements are elements that guide your code between two or more distinct paths.
It begins with a basic true or false structure, where it does something when true and another one if it's false, but feel free to add as many as you feel like.

1: if (x > 3) {
2: x++ -- sums one to the variable
3: } else {
4: x-- -- subs one from the variable
5: }

You don't have to add an "else" if you there's nothing to happen when the statement is nottrue

  • Always open and close the keys "{}"
  • Put the condition inside parenthesis

functions

To keep your code clean and easier to find errors, you can create functions and name them with something that explain its objective. It is even more useful when you have to repeat the same instructions in different moments, so it saves you from writing the same things over and over.

1: function nameFunction (n1, n2) {
2: var sum = n1 + n2 -- sums two numbers
3: return sum -- return the result
4: }
5: console.log(nameFunction(1, 2)) -- shows the result

As you can see, you need to call the function after you create it, else it won't do anything. Inside the parentheses, you have what are called parametes, which are basically variables that work in the function, and you must give the function all of its parameters for it to work. Yoy can create functions with no parameters if not needed, also functions with no return.
